VLSI RTL Project

High-Speed Double-Precision Floating-Point Vedic Multiplier

64-bit floating-point multiplier with hierarchical Vedic mantissa multiplication and dual simulation flow for Icarus Verilog and Vivado xsim.

View Source User Guide

Architecture

Sign XOR, exponent bias-adjust, 53x53 Vedic mantissa multiplier, and normalization stage.

Verification

Self-checking testbench with directed IEEE-754 vectors and automatic pass/fail reporting.
