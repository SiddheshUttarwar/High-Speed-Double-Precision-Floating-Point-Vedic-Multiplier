# Benchmark Plan

## Goal

Track functional and implementation-oriented metrics in a repeatable format.

## Suggested metrics

- Functional pass/fail for directed vectors
- Synthesis timing (Fmax)
- LUT/FF utilization (FPGA flow)
- Dynamic/static power (when available)

## Experiment matrix

- Tool versions: Icarus, Vivado
- Target devices: document in result tables
- Commit hash and date for every run
